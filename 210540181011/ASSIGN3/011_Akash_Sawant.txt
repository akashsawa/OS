1. Write a script where user enters a file path and output says if input path is a file or a directroy

Script:

#!/bin/bash

read -p "ENTER PATH: " path

var=$(echo $path | awk -F"/" '{ print $NF }')


#echo $var
if [ -f $var ]
then
        echo "$var is a file"
else
        echo "directory"
fi

2.Write a script where username is entered and its userid is displayed in the output

Script:
#!/bin/bash

read -p "ENTER USERNAME:  " user

var=$( cat /etc/passwd | grep -i $user | cut -d':' -f3 )

#echo $var
if [ -n $var ]
then
        echo "USERID: $var "
fi
